Keycloak :

Keycloak est un produit logiciel open source permettant la connexion unique avec Identity and Access Management destiné aux applications et services modernes.

Plus précisément dans le cadre de notre application nous utilisons keycloak afin de garantir un niveau de sécurité maximal ainsi qu’une gestion de user optimisé, nous avons conscience que ce choix d’architecture est disproportionné pour ce genre de projet mais nous le voyons aussi et surtout comme un bon moyen de monter en compétence.

SpringBoot :

Spring boot est un framework Java et Java J2EE qui permet d’exposer rapidement et facilement une API RESTful à des services web, qui pourront envoyer des requêtes HTTP à cette dernière.

Dans notre cas nous avons interfacé SpringBoot avec une base de données MySQL.

TypeScript

TypeScript est un langage de programmation libre et open source développé par Microsoft qui a pour but d’améliorer et de sécuriser la production de code JavaScript. Le TypeScript est née d’un constat, " le javascript n’est pas fait pour les projets à grande échelle ".
Typescript est un langage multi paradigmes. On peut faire du fonctionnel, comme de l’orienté objet sans problème. Et nous parlons de vrai orienté objet, pas d’orienté objet via prototype comme en Javascript. La combinaison POO et le fait que Typescript soit un langage fortement typé a tout changé.

Angular

Développé par Google et mis en production en 2009, Angular est une plateforme de développement (framework) open source qui permet de créer des applications web dynamique et immersive.

Angular est utilisé pour créer ce que l’on appelle des “single page applications”, des applications web accessibles via une page web unique, on évite ainsi un chargement de page à chaque nouvelle action ce qui contribue à fluidifier l’application. Le frameword est basé sur une architecture MVC séparant ainsi les données visuelles et les actions pour une meilleure gestion des responsabilité.

Le framework s’appuie sur Typescript une surcouche de JavaScript développé par Microsoft, c’est un langage typé qui permet de créer des classes, des variables et des signature de fonction, il est possible d’utiliser du JavaScript natif avec Angular, mais TypeScript permet de mieux gérer les erreurs.

Il y’a de nombreux framework Js très populaire : Angular, React, Ember, Vue…, les autres framework marchent très bien, sont utilisés sur des site très fréquenté, React et Vue notament. Angular présente aussi un niveau de difficulté supérieur car on utilise du TypeScript plutôt que du JavaScript mais quel sont donc les avantage d’Angular ?

  • Angular est géré par Google, il y’a donc peu de chance qu’il disparaisse et l’équipe de développement du framework est excellente
  • Le TypeScript, ce langage permet un dévelopement beaucoup plus stable, rapide et facile
  • Le framework Ionic, permettant le développement d’application mobile multiplatformes à partir d’une seule base de code utilise Angular
    Bien entendu les autres framework ont aussi leurs avantage mais Angular reste un bon choix pour le developpement frontend
